# 2do Proyecto - Programación Gráfica

Shooter 3D en primera persona estilo *Doom*, desarrollado en **Unity**.

## Descripción

Juego de disparos en 3D con enemigos de distintos tipos (normal, rápido y lento),
sistema de disparo, materiales y escenas propias.

## Estructura

- `Shooter3DDoom/` — Proyecto de Unity (código, escenas, materiales, prefabs).
- `Informe Parcial 2.docx` — Informe del proyecto.
- `enlace del video.txt` — Enlace al video de demostración.

## Requisitos

- Unity (abrir la carpeta `Shooter3DDoom/` desde Unity Hub).

> La carpeta `Library/` no se incluye porque Unity la regenera automáticamente
> al abrir el proyecto.
